## Workflow techniczny

### Pobranie design bundle
```
WebFetch https://api.anthropic.com/v1/design/h/fO9bY5QqPifS9TnXfIn7RA
→ binary gzip 4.3 MB → /tmp/design.tar.gz
→ tar -xzf → /tmp/design-cv/cv-ada/
  ├─ README.md (handoff instrukcje dla agenta)
  ├─ chats/chat1.md (historia 2 iteracji z user)
  └─ project/
     ├─ CV - Adriana Gusciora.html (775 linii — final)
     ├─ CV - Adriana Gusciora v1.html (v1 odrzucone)
     └─ uploads/ (PDF źródłowy + 2 obrazki referencyjne)
```

### Pattern Playwright PDF
- `chromium.launch()` → `newPage()` → `setContent(html, {waitUntil:'networkidle'})` (czeka na Google Fonts CSS)
- `emulateMedia({media:'print'})` żeby aktywować `@media print` reguły z designu
- `pdf({format:'A4', printBackground:true, margin:0, preferCSSPageSize:true})`
- Photo: `fs.readFileSync().toString('base64')` → `data:image/jpeg;base64,...` data URL embedded inline w HTML

### Weryfikacja PDF
- `node -e "fs.readFileSync(...).toString('latin1').match(/\\/Type\\s*\\/Page[^s]/g).length"` → 2 strony ✅
- File size: 3.7 MB (zdjęcie ada2.jpg ~520 KB jako base64 ~700 KB w PDF + assets)
